เกริ่น

ข้อดีของ R คือ มี package ที่พัฒนามากมาย ดู awesome R package เหล่านี้ไม่ได้มากับ base R (package พื้นฐาน ที่มาตอนเราติดตั้ง R โดยจะโหลดให้เราทุกครั้งตอนเวลาเราเปิด R) เราต้องติดตั้ง package ก่อนใช้งาน โดยวิธีปกติติดตั้ง ด้วยคำสั่ง install.packages('pkg') โดยที่ pkg คือ ชื่อของ package ซึ่งต้องอยู่ในเครื่องหมายคำพูด (R เรียกว่า character แต่ภาษาอื่นอาจะเรียกว่า string) จริงๆ แล้วหลังจากติดตั้ง package เราสามารถเรียกใช้ได้เลย โดยใช้คำสั่ง pkg::some_function

ยกตัวอย่างเช่น

# install.packages("ggplot2")
library(ggplot2)
ggplot(mtcars, aes(x=wt, y=mpg)) +
  geom_point()

pacman

package pacman ในที่นี้ย่อมาจาก package management ไม่ได้หมายถึงเกม pacman นะ

# if (!require("pacman")) install.packages("pacman")
library(pacman)
pkgs = c('tidyverse', 'plotly', 'ggthemes', 'hrbrthemes')
p_load(pkgs, character.only=TRUE)
